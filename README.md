# Text-Fabric Workshop in Zurich

This repository contains materials for the workshop on Text-Fabric in Zurich in March and April 2026.

You can install the dependencies with:

```pip install -r requirements.txt```


In the folder make_queries you find the materials for making queries in the BHSA dataset with the TF Browser app.
The file example_queries_bhsa.txt contains example queries for the BHSA dataset that you can run with TF Browser, which you can open with:

```tf etcbc/bhsa```

The file run_queries_with_python.ipynb is a jupyter notebook file which shows how you can run queries with Python. You can open Jupyter Notebook with:

```jupyter notebook```

In the folder visualizations_and_statistics you find a dataset and notebook that we use for the analysis of datasets.
